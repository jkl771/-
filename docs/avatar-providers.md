## 3. “花费价格”怎么看最实用

### 先看计费方式
- **按分钟计费**：最适合视频生成场景
- **按条计费**：适合短视频/单条口播
- **按套餐/点数**：适合持续批量生产

### 我给你的价格判断建议
- **国内优先**：通常比海外更有性价比，且更省网络和接入成本
- **不要只看“有没有免费”**：真正影响成本的是  
  `单条视频花费 = 单价 x 视频时长 x 重试次数`
- **先跑通主链路，再做成本优化**：先选一个稳定API，再逐步替换成更便宜方案
