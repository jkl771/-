## 1. 你现在可以直接看的官方价格页

### D-ID
- 官方价格页：https://www.d-id.com/pricing
- 重点看：
  - 套餐层级
  - credits / 分钟数
  - API 是否包含在套餐里
  - 单月价格与年付价格
- 适合你判断的点：
  - 是否适合“真人驱动 / 数字人口播”
  - 单条视频的 credits 消耗
  - 是否适合长期生产

### 阿里云
- 官网：https://www.aliyun.com/
- 建议直接在站内搜：
  - `数字人 价格`
  - `数字人 API 计费`
  - `视频口播 计费`
- 为什么没直接给你写死金额：
  - 阿里云通常按产品线拆分，不是一个统一定价页
  - 不同能力（形象驱动、口播、合成、API调用）可能分属不同计费项
- 适合你判断的点：
  - 是否国内直连
  - 是否API
  - 是否按分钟计费
